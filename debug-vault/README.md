# Debug Vault

This vault is for plugin development only.

Rules:
- Do not put personal notes here.
- Keep only test files needed for Typsidian development.
- Open this folder directly in Obsidian while running `npm run dev`.

Quick start:
1. `npm install`
2. `npm run setup:vault`
3. (Optional) `npm run setup:hot-reload`
4. `npm run dev`
5. Open `debug-vault` in Obsidian and enable `Typsidian`.
